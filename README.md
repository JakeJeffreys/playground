# Playground

A collection of experiments, demos, and learning projects for exploring different technologies and concepts.

## Projects

### [Kubernetes (k8s/)](k8s/)
A beginner-friendly Kubernetes setup demonstrating core concepts like namespaces, deployments, services, and ConfigMaps. Includes a fully documented nginx deployment with 3 replicas running on minikube.

**Quick start:**
```bash
kubectl apply -f k8s/
kubectl get all -n learning-k8s
```

See the [k8s/README.md](k8s/README.md) for full documentation.

### [AWS Certification (aws-cert/)](aws-cert/)
Comprehensive study materials for the AWS Certified Developer - Associate (DVA-C02) exam. Includes detailed study guides, an 8-week study plan, and a quick-reference cheat sheet covering all exam domains.

**Contents:**
- [Study Guide](aws-cert/README.md) - Complete exam overview and resources
- [8-Week Study Plan](aws-cert/study-plan.md) - Day-by-day study schedule
- [Cheat Sheet](aws-cert/cheat-sheet.md) - Quick reference for limits and concepts

See the [aws-cert/README.md](aws-cert/README.md) to get started.

## About

This repository is a workspace for hands-on learning and experimentation. Each project folder contains its own README with setup instructions and documentation.
